# HewKawJang

## Requirements

- [expo go](https://expo.dev/go)
- node.js
- npm
- docker

## Frontend

Set up frontend project

```bash
cd frontend
npm i
```

Try running

```bash
# When running the project
# Type 'w' to open in web
# Type 'i' to open in IOS sim
npx expo start
```
