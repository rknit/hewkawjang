# HewKawJang

Restaurant reservation application

## Requirements

- node.js
- npm

## How to setup

Setup .env in frontend and backend (refer to the READMEs), then run the following command

```bash
# in hewkawjang directory
npm install
```

## How to run

Run the frontend

```bash
# in hewkawjang directory
npm run frontend
```

Run the backend

```bash
# in hewkawjang directory
npm run backend
```
